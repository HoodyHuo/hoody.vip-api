# hoody.vip web site 

个人网站

> `Vue.js`  
用了[花裤衩的vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)作为模板

> `Springboot` 做后端
 `shiro` 做权限控制 
 `Swagger` 做api调试


> 发布环境 `docker` 整一套, 
DockerFile 构建容器`mysql` `Nginx` `java环境` 
阿里云[容器镜像服务](https://cr.console.aliyun.com) 环境上传,发布
